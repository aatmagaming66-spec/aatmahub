# **App Name**: AATMA HUB

## Core Features:

- AI Gift Matcher tool: A smart assistant that uses simple user input to suggest the best diamond or currency packages for specific gaming servers based on popular trends.
- Modular Gaming Marketplace: A high-density mobile grid display for game titles including MLBB regional variants, Honor of Kings, and Genshin Impact.
- Horizontal Digital Service Carousels: Optimized horizontal scrolling components for browsing OTT services like Netflix and social platforms like Instagram.
- ID Verification Layout: Dedicated input area for Player and Server IDs with visual feedback buttons to simulate identity verification before checkout.
- Contextual Order Tracker: Tab-based UI for users to switch between Pending, Processing, and Completed orders with specialized status indicators.
- Integrated Wallet & Profile Shell: Premium account dashboard displaying balance cards, order statistics, and quick-link navigation settings.
- Persistent App Navigation: A specialized 4-tab mobile bottom bar and a side-drawer navigation menu for rapid accessibility across the store.

## Style Guidelines:

- Primary Color: Vivid Neon Purple (#A140FF) used for buttons, active navigation, and branding highlights.
- Background Color: Deep Night Violet (#0E0911), a heavily desaturated purple variant for the core background to emphasize a premium feel.
- Accent Color: Electric Periwinkle (#8C8CFF), used sparingly for iconography and status text to create high-contrast legibility against the dark base.
- Headline Font: 'Space Grotesk' (sans-serif) for a modern, tech-forward, and gaming-aligned appearance.
- Body Font: 'Inter' (sans-serif) for high readability in small text blocks, pricing labels, and navigation items.
- Mobile-first full-width design with zero horizontal margins on primary banners, utilizing a dense grid and horizontal scroll carousels for digital assets.
- Silky-smooth transitions for side-drawer movement, banner gradients, and button-press feedback using Framer Motion.
